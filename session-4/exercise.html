Exercise

  1. Create a form that takes a number (n) from user, then calculates the sum of all numbers from 1 ... n and displays the sum.
  2. Create a form that takes a name (name) and gender (gender) from the user and displays "Hello Mr. (if gender is male) or Hello Ms. (if gender is female) [name].
  3. Create a form that takes two number from the user (num1) and (num2) and then outputs if the two numbers are divisible "20 is divisible by 2", "19 is not divisible by 2".
  4. Create a form that takes a number from user and creates the mulitplication table for this number from 1 till 12 in the following format "1 * 1 = 1"
  5. Create a user registration form (must contain at least one checkbox, one radio button and one dropdown menu) with validation for all inputs
  6. Create a form that takes data from 10 students which are name and grade, after submission the user is redirected to another page with a table showing the data (name and grade) of the students who passed only (grade >= 50)
