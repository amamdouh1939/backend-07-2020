Example

Create the following classes

  1. Create a base [Animal] class which has the following members $name, $age, and static member $type, create a constructor to instantiate the name and age, create setters and getters for name, age and type, the Animal class has the following methods, walk(), and talk()
  2. Create class [Lion] which inherets from the base [Animal] class and has additional property $numberOfPreys with its setters and getters, and has the following methods, overriden walk() method, and overriden overloaded talk(), talk($string) methods
  3. Create class [Bird] which inherets from the base [Animal] class and has additional property $distanceCovered with its setters and getters, and has the following methods, overriden walk() method, and overriden overloaded talk(), talk($string) methods
